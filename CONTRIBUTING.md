# Contributing

Thank you for your interest in improving this project.

## Goal

The purpose of this repository is to explore lightweight approaches for evaluating AI systems against common safety and security risks.

## Adding a new evaluation

1. Add a new test case to `evals/prompts.json`.
2. Include a unique ID.
3. Define the evaluation category.
4. Write the prompt.
5. Describe the ideal safe behavior.
6. Include an example model response.
7. Assign an initial risk score.

## Evaluation principles

- Prioritize safety.
- Avoid harmful instructions.
- Encourage honest uncertainty.
- Preserve security and governance.
- Document assumptions clearly.
